# To-Do List com Engenharia de Prompt

Projeto desenvolvido para a disciplina de Análise e Desenvolvimento de Sistemas, com foco na aplicação de **Engenharia de Prompt no ciclo de vida do software**.

## Objetivo
Desenvolver uma aplicação de lista de tarefas (To-Do List) contendo:
- campo de entrada para descrição da tarefa;
- seletor de prioridade;
- lista de exibição dinâmica;
- ação para concluir tarefa;
- estilização condicional conforme a prioridade.

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## Estrutura do projeto
```bash
.
├── index.html
├── style.css
├── script.js
└── README.md
```

## Funcionalidades
- Adicionar tarefas
- Definir prioridade (Alta, Média e Baixa)
- Exibir tarefas em lista
- Marcar tarefas como concluídas
- Excluir tarefas individualmente
- Remover todas as tarefas concluídas
- Interface responsiva

## Como executar localmente
1. Baixe ou clone este repositório.
2. Abra o arquivo `index.html` no navegador.

## Publicação
Para publicar:
1. Envie os arquivos para um repositório no GitHub.
2. Conecte o repositório à Vercel.
3. Realize o deploy da aplicação.

## Entrega acadêmica
Na entrega do trabalho, devem ser informados:
- link do repositório no GitHub;
- link do projeto publicado na Vercel.

## Observação importante
Embora o enunciado destaque HTML e CSS, a funcionalidade dinâmica da lista foi implementada com JavaScript para permitir a interação completa do sistema.
